== Description ==
It is only for this plugin: Welcart e-Commerce.
A narrowing search is possible in price range.

When enabled, the menu "Welcart Shop", Price Search will be added.


== Installation ==
1. Upload the entire `price-search-w` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
